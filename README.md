# Obfuscator Studio ✨

> 面向前端 / 小游戏 / 小程序团队等的 JS/TS 一站式混淆加固工作台。

[![](https://img.shields.io/badge/License-ISC-4caf50)](#)

Obfuscator Studio 提供现代化图形界面，支持批量文件与整项目模式，可组合十余种混淆策略，实时回显进度与结果，帮助你快速保护核心业务代码。

## 功能亮点

| 功能 | 描述 |
| --- | --- |
| 双模式入口 | 文件模式/项目模式一键切换，支持 `.js/.ts/.tsx` |
| 输出策略 | 覆盖原文件或“另存为”，保留原有目录结构 |
| 高级混淆 | 控制流平坦化、字符串数组、死代码注入、Unicode 转义等 |
| TypeScript 支持 | 内置 `esbuild` 先转译 TS/TSX，再进行混淆 |
| 实时日志 | 任务进度、成功/失败统计即时呈现，可一键打开输出目录 |

## 环境要求
- Windows 10 x64（可自行扩展 macOS/Linux）

## 图形界面使用流程

1. **选择目标模式**
   - `文件模式`：点击“选择 JS/TS 文件”，支持多选。
   - `项目模式`：选择包含脚本的目录，工具会自动递归匹配。
2. **配置输出策略**
   - `覆盖原文件`：混淆结果写回原路径（TS/TSX 自动改写为 `.js`）。
   - `另存为`：指定输出目录，工具保持与源目录一致的层级。
3. **调整混淆选项**
   - 勾选紧凑输出、控制流平坦化、对象键转换等开关。
   - 在“保留名称/字符串”文本框中逐行添加需要跳过的标识符或字面量（支持正则）。
4. **执行混淆**
   - 点击“开始混淆”，实时日志会展示处理状态。
   - 成功/失败统计会在任务结束后给出，若选择另存为可使用“打开输出目录”快速查看。

## 常见问题

- **提示“请至少选择一个 JS 文件”**：文件模式必须先完成文件选择；项目模式同理需要目录。
- **另存为模式无输出**：确保已指定“输出目录”；若目录无写入权限，请换到其他位置。
